# HireBest

AI Resume Screener for Hiring Teams ‚Äî Score 100 CVs in 38 Seconds.

Marketing site rebuilt in Vite + React 19 + react-router-dom + Tailwind v4.

## Dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run preview
```

## Stack

- Vite 6
- React 19
- react-router-dom 7
- Tailwind CSS v4 (via @tailwindcss/vite)
- lucide-react icons
- TypeScript
